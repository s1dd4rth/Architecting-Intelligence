The Brain

Structured Prompting

The Golden Formula: Role + Context + Task + Constraints.

def get_prompt(user_input):
    role = "Act as a Senior Python Architect."
    context = "User is a junior dev."
    task = f"Explain this concept: {user_input}"
    const = "Use analogies. Max 3 sentences."
    
    return f"{role}\n{context}\n{task}\n{const}"

The Brain

Context Window Management

The "RAM" of the AI. If you feed it too much history, it crashes (or forgets). You must prune.

chat_history = []
MAX_TOKENS = 8000

def add_message(msg):
    chat_history.append(msg)
    
    # Simple FIFO Truncation
    estimated_tokens = sum(len(m)/4 for m in chat_history)
    
    while estimated_tokens > MAX_TOKENS:
        # Remove oldest message (Forget)
        removed = chat_history.pop(0)
        estimated_tokens -= len(removed)/4
        print("Memory Full. Forgetting oldest...")

The Soul

Vector Database

A specialized database (Pinecone, Chroma) to store and search embeddings instantly.

# Simulating a Vector Store
vector_store = []

def add_document(doc_id, text):
    vec = get_vector(text)
    vector_store.append({"id": doc_id, "vector": vec, "text": text})

def search(query):
    query_vec = get_vector(query)
    # Compare query against ALL docs
    scores = [np.dot(doc["vector"], query_vec) for doc in vector_store]
    best_idx = np.argmax(scores)
    return vector_store[best_idx]

Flow Engineering

Rigid, reliable chains. Unlike Agents, Flows don't "think"‚Äîthey follow a script.

    # Step 1: Outline
    outline = generate(f"Outline a post about {topic}")
    
    # Step 2: Draft
    draft = generate(f"Write post based on: {outline}")
    
    # Step 3: Critique
    critique = generate(f"Critique this draft: {draft}")
    
    # Step 4: Finalize

Autonomous Agents

The Agency

Ralph Wiggum Loops

Anti-Pattern: The agent keeps "Trying" but getting nowhere. You need detection logic. 

previous_actions = []

def detect_loop(new_action):
    previous_actions.append(new_action)
    
    # Check if last 3 actions are identical
    if previous_actions[-3:] == [new_action]*3:
        raise Exception("LOOP DETECTED! Terminating.")

# Inside Agent Loop:
try:
    detect_loop(current_action)
except:
    force_human_handoff()

def manager_ai(user_request):

The Team

Parallel Execution

Running multiple agents at once to reduce latency.

import asyncio

async def plan_trip_async():
    
    # Start both agents simultaneously
    task1 = flight_agent.search("Paris")
    task2 = hotel_agent.search("Paris")
    
    # Wait for both to finish
    flight, hotel = await asyncio.gather(task1, task2)
    
    return combine(flight, hotel)
